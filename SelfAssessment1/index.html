Inline Font Sizes (absolute + relative)

  • 12px — great for tight UI labels (e.g., compact badges).
  • 16px — readable body text on desktop; common browser default.
  • 1.25rem — scales with root size; good for section headings.
  • 0.875em — relative to parent; nice for fine print inside a card.
  • 110% — subtle emphasis within a component without hard-coding px.
  • 3vw — fluid hero headline that grows/shrinks with viewport width.

Positioning Playground

Examples of static, relative, absolute, and fixed. Footer stays fixed at the bottom of the screen.

Static — default document flow (articles, paragraphs).
Relative — nudged from normal spot; also establishes a containing block for absolute children.
Absolute — positioned relative to this section (nearest positioned ancestor); great for badges/overlays.
Fixed — pinned to the viewport (e.g., “back to top” or floating CTA).

Important <head> Tags for SEO

  • <title> — concise, unique page title.
  • <meta name="description"> — compelling summary for snippets/CTR.
  • <link rel="canonical"> — preferred URL to handle duplicates.
  • <meta name="robots"> — index/follow, or control crawling per page.
  • <meta name="viewport"> — mobile-friendly rendering.
  • Open Graph / Twitter Cards — better social sharing previews.
  • Structured data (<script type="application/ld+json">) — eligibility for rich results.

  3. <h1>..<h6> — headings hierarchy

  5. <a> — hyperlinks / anchors
  6. <section> — semantic grouping of content
  7. <nav> — navigation landmarks
  8. <footer> — page/footer content
  9. <img> — images
  10. <ul>, <ol>, <li> — lists
